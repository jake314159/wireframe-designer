Wireframe Designer
===================

A very very basic wireframe design tool.


Features
--------
+ Multiple wireframe boxes
+ Text labels
+ Background designs & colours
+ PNG export


![Creating a wireframe](/exampleImages/UI/making_a_basic_wireframe_ubuntu.png)
